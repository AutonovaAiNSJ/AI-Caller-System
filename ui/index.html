Quick-Start Setup
  1. Open Supabase Dashboard → SQL Editor → paste and run supabase_schema.sql
  2. Go to ⚙️ Settings → LiveKit → enter URL, API Key, API Secret → Save
  3. Go to Settings → Gemini → enter Google API Key, select model & voice → Save
  4. Go to Settings → Vobiz SIP → enter domain, username, password, phone → Save → click ⚡ Create SIP Trunk
  5. Go to ✏️ AI Prompt → customise for your business → Save
  6. Go to 🤖 Agents → create at least one agent profile
  7. Go to Single Call - enter your own number - click Place Call
  8. Hear the AI speak within 5–10 s of the call connecting OK
